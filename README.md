# 🧠 Stroke_Prediction: AI Diagnostic Assistant

**Stroke_Prediction** ek advanced Machine Learning web application hai jo clinical data (vitals) ka use karke patients mein **Stroke** aur **Heart Disease** ke risk ko predict karta hai. Isse healthcare professionals aur individuals ko early warnings mil sakti hain.

## 🚀 Key Features
* **Dual-Analysis Mode**: Stroke (Neurological) aur Heart Disease (Cardiac) dono ke liye alag-alag AI models.
* **Real-time Prediction**: Flask backend ke zariye fast processing aur accurate results.
* **Professional Reports**: Analysis ke baad ek detailed diagnostic report generate hoti hai.
* **Responsive UI**: Modern interface jo mobile aur desktop dono par smooth chalta hai.

## 🛠️ Tech Stack
* **Language**: Python 3.10+
* **Framework**: Flask
* **Machine Learning**: Scikit-learn (Random Forest, Logistic Regression)
* **Frontend**: Tailwind CSS, JavaScript (Vanilla)
* **Data Handling**: Pandas & NumPy

## 📊 How it Works
1. User patient ke vitals input karta hai (e.g., BMI, Glucose Level, Ejection Fraction).
2. Backend par trained `.pkl` models data ko process karte hain.
3. System risk level (High/Low) predict karke ek dynamic report page par redirect kar deta hai.
